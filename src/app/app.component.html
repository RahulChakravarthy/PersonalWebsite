Paint it! Sound Classification

  • Built the Tensorflow urban sound classification model to classify sounds produced by a popular south asian percussion instrument
  • Model is trained via accessing stored raw labeled sound data from an Amazon S3 bucket
  • Utilized Tensorflow Serving to expose a rest API between model and server application

Paint it! Web Application

  • Built the full stack MEAN application of Paint it! to interact with the urban sound classification neural network
  • The application supports sound recording and uploading to Amazon S3 buckets via Socket.io communication
  • Utilized sockets to architect the classification interface between the tensorflow model and the client side application
  • Classified sounds get transformed into visual drawings via D3.js and displayed to the screen

Paint it! Android Application

  • Built the Android client application for Paint it! in an MVVM architecture
  • Utilized Amazon Cognito for user identification along with Android components to allow for easily scalable and decoupled code

ClassLink

Captcha Cracker

  • Built a captcha parsing program that detects alpha numeric text in images and outputs the entire string
  • Used python image library to convert the image into RGB codes and pattern analysis to identify all the characters

Hand Gesture 2048
